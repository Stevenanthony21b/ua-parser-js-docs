# `cpu.architecture`

Type of the CPU architecture. 

- `68k`
- `amd64`
- `arm`
- `arm64`
- `armhf`
- `avr`
- `ia32`
- `ia64`
- `irix`
- `irix64`
- `mips`
- `mips64`
- `pa-risc`
- `ppc`
- `sparc`
- `sparc64`

::: tip
**Conventions**

- 32-bit version of **'x86'** is referred as `ia32`
- 64-bit extension of **'x86'** (also known as **'x86-64'** or simply **'x64'**) is referred as `amd64`.
:::
